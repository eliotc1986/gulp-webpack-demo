## Gulp/Webpack example

Use npm modules with client side projects

## Dependencies

### npm

Built with npm version `12.13.0`

### gulp

```
$ npm install --g gulp-cli
```

## Setup

From the project root, run:

```
$ npm install
```

And then:

```
$ npm start
```

## For minified assets

Run:

```
$ npm run build
```
